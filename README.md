# Cura Wanhao Duplicator I3 Plus
Cura 2.3+ support for the Wanhao Duplicator i3 Plus.

Inspired by [cura-wanhaoduplicatori3 repo](https://github.com/johnrees/cura-wanhaoduplicatori3) which supports older (<=2.1) versions of Cura.

## Installation

### Windows

1. Copy `wanhao-backplate.png` to `%programfiles%\Cura 2.3\resources\images` (or similar)

2. Copy `wanhao_duplicator_i3_plus.def.json` to `%programfiles%\Cura 2.3\resources\definitions`

3. Restart Cura

4. Add Printer > Other > Wanhao Duplicator I3 Plus
